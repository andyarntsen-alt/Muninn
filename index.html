HVORDAN DET FUNGERER
Tre steg. Ti minutter.
01

Klon og bygg

git clone github.com/andyarntsen/muninn
npm install && npm run build

Krever Node.js 18+ og Claude CLI.

02

Kjør setup

npm run setup

Interaktiv veiviser. Kobler Telegram, setter opp AI-backend. Ferdig.

03

Snakk

npm start

Terminal eller Telegram. Han husker alt fra dag én.
